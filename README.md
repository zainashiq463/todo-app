# ✅ Todo App

A simple and clean Todo App built with **React** and **Vite**.

---

## 🚀 Live Demo

> _[Add your Vercel link here once deployed](https://todo-app-six-azure-44.vercel.app/)_

---

## ✨ Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter by **All / Active / Done**
- See remaining task count

---

## 🛠️ Tech Stack

- React
- Vite
- CSS

---

## 📦 Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Then open the link shown in the terminal in your browser.

---

## 🎯 Concepts Practiced

- React components
- `useState` hook
- Event handling
- Conditional rendering
- Array filtering & mapping
